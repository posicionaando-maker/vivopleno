# 🌿 Vivo Pleno - Cultiva una vida plena cada día

## ¿Qué es?
Una PWA (Progressive Web App) basada en psicología positiva, Terapia Cognitivo-Conductual, mindfulness y hábitos atómicos. Te ayuda a construir una vida plena mediante consejos diarios y pequeñas tareas alcanzables.

## Características
- ✅ Consejos diarios rotativos según 7 pilares del bienestar
- ✅ Banco de tareas + tareas personalizables
- ✅ Seguimiento de racha, radar de pilares y mapa de calor
- ✅ Sección de logros destacados para guardar reflexiones
- ✅ Exportar/importar datos para respaldo
- ✅ Offline first (funciona sin internet después de instalar)
- ✅ Instalable en móvil como app nativa
- ✅ Recursos de ayuda para Cuba

## Instalación en móvil (Android/iPhone)

1. Abre esta página en **Chrome** (Android) o **Safari** (iPhone)
2. Toca el menú (3 puntos o compartir)
3. Selecciona **"Agregar a pantalla de inicio"** o **"Instalar app"**
4. Usa la app desde el icono en tu escritorio

## Cómo usarla

### Cada día:
1. Abre la app y lee el **consejo del día**
2. Elige 3-5 tareas de la lista (o agrega las tuyas)
3. Completa las que puedas y márcalas con ✓
4. Guarda tus **logros** o reflexiones especiales

### Seguimiento:
- Revisa tu **progreso** para ver el radar de pilares y mapa de calor
- Exporta tus datos desde Ajustes para respaldarlos
